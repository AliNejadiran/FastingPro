# 🔥 Fasting Pro

A modern fasting and fitness tracker built with Python & Tkinter.  
Track your daily water intake, meals, workouts, and fasting progress — all in one simple desktop app.

## 🚀 Features
- 💧 Water tracker (0/8 glasses)
- 🍎 Meal logger (0/3 meals)
- 💪 Workout logger (0/2 sessions)
- ⏳ Real‑time fasting progress bar
- 📊 Daily report with smart tips
- 💾 Auto‑save data (JSON)

## 🖥️ Run from source
```bash
git clone https://github.com/AliNejadiran/FastingPro.git
cd FastingPro
pip install -r requirements.txt
python fasting_pro.py
```
